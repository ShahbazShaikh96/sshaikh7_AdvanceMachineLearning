# Advanced Machine Learning Assignments

This repository contains applied deep learning and advanced machine learning assignments organized by topic. Each folder includes the original notebook/report files plus a short README explaining the project purpose, methods, and files.

## Repository Structure

| Folder | Assignment Name | Main Work |
| --- | --- | --- |
| `01_IMDB_Sentiment_Neural_Network_Comparison` | IMDB Sentiment Classification: Neural Network Architecture and Regularization Comparison | Dense neural networks, activation/loss comparisons, dropout, L2 regularization |
| `02_Cats_vs_Dogs_CNN_vs_VGG16_Transfer_Learning` | Cats vs Dogs Image Classification: CNN From Scratch vs VGG16 Transfer Learning | Custom CNNs, pretrained VGG16, training-size comparisons |
| `03_Jena_Climate_Time_Series_Forecasting_RNNs` | Jena Climate Forecasting: RNN, GRU, LSTM, and Conv1D-GRU Comparison | Time-series forecasting, sequence windows, recurrent architectures |
| `04_IMDB_Text_Classification_Learned_vs_GloVe` | IMDB Text Classification: Learned Embeddings vs Pretrained GloVe Embeddings | Text classification, learned embeddings, pretrained word vectors |
| `05_Efficient_CNN_Digit_Classification` | Efficient CNN Development for Handwritten Digit Classification | CNN workflow, monitoring, callbacks, hyperparameter comparison |

## Main Themes

- neural network architecture comparison
- sentiment classification
- convolutional neural networks
- transfer learning
- recurrent neural networks for time-series forecasting
- pretrained word embeddings
- training monitoring and hyperparameter comparison
